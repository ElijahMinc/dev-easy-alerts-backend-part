# .prettierrc

1. Open settings.json
2. "prettier.configPath": "./backend/.prettierrc"

## Available Scripts

In the project directory, you can run:

### by the first: `yarn install`

### `yarn dev` for dev mode

### `yarn build` for build project

### `yarn start` for run project after build

### OR RUN TWO COMMANDS `yarn "start:build` for run project after build
